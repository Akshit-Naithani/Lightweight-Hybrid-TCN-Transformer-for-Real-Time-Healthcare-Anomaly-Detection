Lightweight Hybrid TCN-Transformer for Real-Time Healthcare Anomaly Detection
=============================================================================
Implementation of the model described in:
"A Lightweight Hybrid Temporal CNN-Transformer Architecture for Real-Time
Healthcare Anomaly Detection on Wearables"

Author: Based on the paper by Akshit Naithani and Vrishin Jain

Date: 2026-03-01

The model combines a Temporal Convolutional Network (TCN) block for efficient
local feature extraction with a factorized self-attention mechanism (Linformer)
to capture long-range dependencies. The design is tailored for deployment on
resource-constrained microcontrollers (TinyML).
